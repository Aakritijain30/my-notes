# 📝 My Notes App

> A clean, minimal note-taking app built with pure HTML, CSS & JS. Create, search, and manage notes instantly. Move notes to trash, restore or permanently delete them. Data saved in localStorage — no login, no server needed. Just open and start writing!

---

## 🌐 Live Demo

### 👉 [Click here to try the app](https://Aakritijain30.github.io/my-notes)

---

## 🌟 Features

- 📌 **Create Notes** — Add notes with a title and body instantly
- 🔍 **Real-time Search** — Filter notes as you type
- 🗑️ **Trash System** — Soft delete with restore option
- ❌ **Permanent Delete** — Remove notes forever when needed
- 💾 **localStorage** — Notes persist even after page refresh
- ⌨️ **Keyboard Shortcut** — `Ctrl + Enter` to save a note
- 📱 **Responsive Design** — Works on mobile, tablet & desktop
